# 👩‍💻 Design Library: Admin
The Design Library provides a place for administrators of instances to quickly and easily get up-to-speed on how pages are constructed and how all of the functionality works.

As an admin, you may be interested in: 

- Configuring for your users
- Submitting bug reports and feature requests

The fastest way to get started is to:

1. Go through the pages listed in the menu
1. Familiarise yourself with the various layouts
1. Check out the components that build each page
1. ...
